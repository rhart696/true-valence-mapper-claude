Toast Notification System Demo

Basic Toasts

Click any button above to see a toast notification appear in the top-right corner. The toast will automatically dismiss after 3 seconds, or you can click the X button or press Escape to dismiss it manually.

Custom Durations

showSuccess('Message', 1000); // Custom duration
showToast('Message', 'info', 0); // Manual dismiss only

Multiple Toasts

The system supports up to 5 visible toasts at once. Additional notifications are queued and appear as others dismiss. Try stacking multiple notifications to see how they're managed.

Real-World Examples

Keyboard Shortcuts:
• Tab - Navigate between close buttons
• Escape - Dismiss the most recent toast
• Enter or Space - Activate close button
