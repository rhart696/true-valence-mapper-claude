## Prerequisites Before Proceeding

### Must-Have Answers

1. **Insurance Coverage**
   - Talk to insurance company THIS WEEK
   - Will they cover workplace relationship tool?
   - What's premium increase?
   - If no coverage or huge premium: **STOP**

2. **Project Owner**
   - Named individual (not "the team")
   - 15-20 hours/week protected time
   - Authority to make decisions
   - If no owner: **DON'T START**

3. **External Advisor**
   - Makes go/no-go decisions
   - Prevents sunk cost bias
   - Could be: consultant, board member, external coach
   - If no advisor: **BIAS WILL CREEP IN**

4. **The Miro Question**
   - "Why not Miro + templates?"
   - Need SPECIFIC answer, not vague "ours is better"
   - If can't answer: **USE MIRO**

5. **Risk Tolerance**
   - OK spending $6-9K to learn "don't build"?
   - Commit to honest decision based on data?
   - Accept NO-GO as success (avoided $28K waste)?
   - If not OK with this: **DON'T START**
